## screen

```sh
# ssh into remote server
ssh remote-server

# start screen
screen

# screen help
<c-a>?

# create a new window
<C-a>c

# Switch between windows by number
<C-a>0
<C-a>1..9

# Switch
<C-a>n # to next
<C-a>p # or previous window

# Create a new window in split
<C-a>| && <C-a><Tab> && <C-a>c # Vertical
<C-a>S && <C-a><Tab> && <C-a>c # Horizontal

# Close all splits
<C-a>Q
```
